Yatzy Simulator

Heuristic-based algorithm for Yatzy:

For each of the 15 combinations that are possible to aim for:

  1. Estimate an expected score if aiming solely for that.
  2. Subtract the cost function. (The cost function represents what you hope to get for each combination, kind of.)
  3. Multiply this difference by the weight function. This helps to prioritize e.g. the upper half, to get the bonus.

The max value determines which combination to aim for (and which dice to save). View on GitHub

It scores decently, probably better than most humans, but sometimes makes stupid decisions. Average score: 229
